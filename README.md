# Holmes Environment Automator 🛠️
**Instant, zero-touch developer environment configuration for the modern engineer.**

Manual environment setup is the "death by a thousand cuts" for developer productivity. Between configuring compilers, resolving dependency paths, and managing dual-boot friction, engineers lose hours of high-leverage time. 

**Holmes Automator** abstracts the chaos into a single, executable payload. Select your stack, generate your script, and let the machine handle the rest.

## 🚀 Live Application
Access the production engine at: [maa-tpa.com](https://maa-tpa.com)

## ✨ Core Features
- **One-Click Toolchains:** Automate the installation of GCC, Clang, CMake, and build tools.
- **Cross-OS Logic:** Tailored payloads for Windows (PowerShell) and Arch Linux (Bash).
- **Silent Installation:** Utilizes `winget` and `pacman` with no-confirm flags for unattended setup.
- **High-Performance Architecture:** Lightweight, dependency-free script generation.

## 🛠️ Built With
- **PowerShell & Bash:** The core automation logic.
- **JavaScript:** Client-side script concatenation.
- **Cloudflare:** Enterprise-grade security and global edge delivery.
